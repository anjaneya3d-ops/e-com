Upload your Meesho or Flipkart files

Drag the original files straight from your seller panel — no need to rename columns. The app understands Meesho's format directly.

📦

Meesho Orders CSV

From Supplier Panel → Orders → Download. File name starts with Orders_

💰

Meesho Payment XLSX

From Payments → Outstanding Payments → Download. Includes settlements, fees, returns

🛒

Flipkart Orders CSV

From Seller Hub → Orders → Export. Auto-detected from column headers

📝

Generic CSV

If your file has columns: order_id, date, platform, product, price, cost, quantity, status

💡 How the matching works

When you upload both the Orders CSV and the Payment XLSX, the app joins them using Sub Order No. You'll then see settlement amounts, commission, and fees for each order — that's how outstanding payments are tracked.

Payment ledger

Settlement details from your Meesho payment file. Negative settlements indicate net deductions (e.g. returns where shipping cost exceeded sale).

Auto-fetch via API (optional)

For real-time order pulls. Most users won't need this — uploading the CSV/XLSX once a day works fine.

Flipkart Marketplace API

How to get credentials
  1. Log in to Flipkart Seller Hub
  2. Settings → API Access → request credentials
  3. Approval takes 1–3 business days

Meesho Supplier API

How to get credentials
  1. Email supplier-api@meesho.com from your registered seller email
  2. Mention your supplier ID and use case
  3. They'll send you an API token if approved

⚠ Security

Never share your seller password. Only use API tokens generated from the seller panel. Backend must be running on port 3001 (cd backend-example && npm install && npm start).
