Connect your seller data

Choose how you want to get orders into the tracker. You can use both — start with CSV today, set up API later.

1

Auto-fetch via API

Real-time · Needs setup · Best for ongoing use

Pulls upcoming orders automatically every time you click Fetch. Requires API credentials from each platform.

Flipkart

How to get credentials
  1. Log in to Flipkart Seller Hub
  2. Settings → API Access → request credentials
  3. Approval takes 1–3 business days

Meesho

How to get credentials
  1. Email supplier-api@meesho.com from your registered seller email
  2. Mention your supplier ID and use case
  3. They'll send you an API token if approved

Backend required: Run cd backend-example && npm install && npm start first. The backend handles auth securely on port 3001.

2

Upload CSV file

No setup · Works today · Best to start with

Download your order report from the seller panel and upload it here. No credentials needed.

Flipkart

  1. Open Seller Hub → Orders → Manage Orders
  2. Filter by status (e.g. Approved, Packed, Dispatched)
  3. Click Export to Excel and save as CSV

Meesho

  1. Open Supplier Panel → Orders
  2. Pick date range or status filter
  3. Click Download and convert to CSV

Select a CSV file from your computer:

CSV columns expected: order_id, date, platform, product, price, cost, quantity, status. If your platform export uses different column names, rename them or use the CSV import tab to paste and review first.

⚠ Security note

Never share your seller login password with any tool. Only use API tokens generated from the seller panel — they can be revoked anytime if compromised. Credentials are saved in your browser's localStorage, which is fine for local personal use but unsafe to deploy publicly.
